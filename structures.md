
# 项目结构目录

```
├── config.js
├── controllers
├── db.js
├── log
├── model
├── package.json
├── routes
├── server.js
├── static
├── styles.xml
├── tcp
├── tcpSocket.js
├── test
├── utils
└── websocket.js
```

#### config.js

用于存放一些敏感的配置信息，不上传至github。

#### controllers

处理主要业务逻辑

#### db.js

mongodb数据库相关

#### log

日志文件

#### model

数据库模型

#### routes

路由信息

#### server.js

服务器主文件

#### static

静态目录，用于存放上传的图片。

#### styles.xml

导出excle表格所需配置文件

#### tcp

处理tcp socket相关函数

#### tcpSocket

tcp socket 核心文件

#### test

测试文件目录

#### utils

一些共用函数

#### websocket

处理websocket的核心文件
